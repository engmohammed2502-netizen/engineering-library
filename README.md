#  مكتبة كلية الهندسة - جامعة البحر الأحمر

##  نظرة عامة
نظام إلكتروني لمكتبة كلية الهندسة يسمح بإدارة المواد الدراسية لكل التخصصات والسمسترات.

##  المميزات
- تسجيل دخول بأنواع مختلفة (طالب، دكتور، رووت، ضيف)
- 5 تخصصات هندسية
- 10 سمسترات لكل تخصص
- رفع وتنزيل الملفات (PDF, DOC, PPT, صور)
- منتدى نقاش لكل مادة
- نظام صلاحيات متكامل
- تصميم متجاوب للجوال

##  التركيب السريع

### المتطلبات المسبقة
- Node.js 16+
- MongoDB 4.4+
- Nginx (للإنتاج)
